# virgate_pipleine_v1
Stage one of python pipelines project for Virgate
## usage 
1. clone the repo 
2. set the following system envs 
  - `sharepoint_email` 
  - `sharepoint_password`
  - `sharepoint_site_name`
  - `sharepoint_url_site`
  - `sharepoint_doc_library`
3. create a cron job for `redirect.py`
